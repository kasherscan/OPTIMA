# DanceFlow - by Optima

Application de gestion d'ecole de danse.

## Deploy sur Vercel

1. Push ce dossier sur GitHub
2. Va sur vercel.com
3. "New Project" > importer le repo GitHub
4. Vercel detecte automatiquement React
5. Clique "Deploy"
